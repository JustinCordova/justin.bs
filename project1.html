Lexical Analyzer

A lexical analyzer is a program that breaks down a given text into smaller parts, such as words or sentences. It identifies and categorizes these parts based on their grammar, syntax, and semantics. This specific lexical analyzer is desigend for a Mini C-Like (MCL) language. The ebnf for this langauge can be found below. This project was created in c++. The project was developed in 3 different stages. The first stage is the lexer which takes text and converts them into tokens based on the ebnf. The second stage is the parser which uses the tokens generated by the lexer and checks their syntax line by line. The last stage utilizes overloaded operatores to check the values of the tokens with the additional use of stage 1 & 2.
